# 🎓 منصة مستشار الثانوية العامة والجامعات الأهلية الذكية

منصة ويب تفاعلية ومتكاملة مصممة خصيصاً لمساعدة طلاب الثانوية العامة بمختلف مراحلها والدبلومات الفنية والأزهر الشريف في مصر خلال فترة التنسيق الحساسة لعام 2026. تهدف المنصة إلى تسهيل عملية اتخاذ القرار واختيار المستقبل الدراسي المناسب بكل سهولة وأمان وبشكل مجاني تماماً.

---

## 🌟 مميزات المنصة الرئيسية:

*   **🧮 حاسبة المجموع الذكية:** لحساب النسبة المئوية فوراً بمجرد إدخال المجموع بالدرجات (من 410 أو أي مجموع مخصص) مع تحديد الشعبة (علمي علوم، علمي رياضة، أدبي، أو دبلومات).
*   **🏫 دليلك للجامعات الأهلية والخاصة:** قاعدة بيانات تفاعلية تضم الجامعات الأهلية والمعاهد المتاحة في مصر، مع توضيح التنسيق التقريبي والمصروفات السنوية لكل كلية وسهولة البحث والفلترة.
*   **📝 أداة ترتيب الرغبات:** أداة مرنة تسمح للطالب بترتيب رغباته المفضلة وحفظها كقائمة استرشادية، مع إمكانية نسخها ولصقها مباشرة في موقع التنسيق الإلكتروني.
*   **🤖 مستشار الذكاء الاصطناعي (AI Chatbot):** مساعد ذكي متكامل مدعوم بنموذج **Gemini API** للإجابة على تساؤلات الطلاب حول التخصصات وسوق العمل وتوجيههم أكاديمياً وقانونياً (شروط تقليل الاغتراب والتوزيع الجغرافي).

---

## 🔒 حماية وخصوصية الـ API Key:

لحماية خصوصية وأمان مستخدمينا ومطورينا، **لا تطلب المنصة ولا تخزن أي مفاتيح سرية على خوادم عامة**.
*   يتم تشغيل مستشار الذكاء الاصطناعي عن طريق إدخال المستخدم لـ **Gemini API Key** الخاص به في واجهة الموقع من خلال زر **"اضبط مفتاح الذكاء الاصطناعي"** الأنيق في أعلى الصفحة.
*   يتم حفظ المفتاح بأمان محلياً داخل متصفح المستخدم (**LocalStorage**) ولا يتم رفعه أو مشاركته مع أي جهة خارجية، بل يتم توجيه الاستدعاءات مباشرة من متصفحك لخادم Google الآمن.

---

## 📂 الملفات التي تم تعديلها وتطويرها:

لتأمين مفتاح الـ API وتوفير أفضل تجربة استخدام مرنة وخالية من الثغرات، قمنا بتعديل وتحديث الملفات التالية:

1. **`src/lib/gemini.ts`** (ملف جديد):
   * يحتوي على دوال إدارة مفتاح الـ API الخاص بـ Gemini وتخزينه في الـ `LocalStorage`.
   * يتضمن دالة الاستدعاء المباشر لخوادم Google بأمان تام دون المرور بخادم وسيط عند توفر المفتاح محلياً.

2. **`src/components/ApiKeyManager.tsx`** (مكون جديد):
   * واجهة إعدادات أنيقة منبثقة للتحكم بالمفتاح (إدخال، إظهار/إخفاء، حفظ محلي، وحذف).
   * تحتوي على رابط مباشر للحصول على مفتاح API مجاني بالكامل من Google AI Studio، وتنبيهات تفاعلية ذكية لحالة الاتصال.

3. **`src/App.tsx`**:
   * دمج مدير مفتاح الـ API في الهيدر الرئيسي (Navigation Bar) ليكون واضحاً للطلاب دون تشتيت الانتباه.

4. **`src/components/AiRanker.tsx`**:
   * تعديل منطق العمل للتحقق من وجود مفتاح الـ API المدخل محلياً، وإذا توفر يتم استدعاء نموذج **Gemini 3.5 Flash** مباشرة من المتصفح لمعالجة صور الدرجات وترتيب الرغبات؛ وفي حال عدم توفره يتم التراجع التلقائي للاتصال بالخادم الداخلي أو مطالبة المستخدم بوضع المفتاح عبر فتح صندوق الإعدادات تلقائياً.

5. **`src/components/DiplomaAdvisor.tsx`**:
   * تحديث الشات بوت الداخلي لدعم الاستدعاء المباشر بـ API Key العميل من المتصفح مع الحفاظ على خصوصية المحادثة.

6. **`src/components/ResultsInquiry.tsx`**:
   * دمج الأخبار العاجلة ومؤشرات التنسيق الحية عبر الذكاء الاصطناعي من خلال الاستدعاء المباشر والمحمي بمفتاح المستخدم.

---

## 🛠️ التقنيات المستخدمة:

*   **React 18 & TypeScript** (لتطوير واجهة مستخدم قوية، سريعة، ومكتوبة بكفاءة عالية).
*   **Tailwind CSS** للتصميم العصري المتجاوب بالكامل مع كافة الشاشات والأجهزة اللوحية والمحمول.
*   **Motion (Framer Motion)** لإضافة حركات انتقال وانبثاق ناعمة ومحترفة تعزز من تجربة الاستخدام السلسة.
*   **Google Gemini API** لتشغيل المساعد الذكي والتحليل التلقائي.

---

## 🚀 طريقة التشغيل والنشر مجاناً على GitHub Pages:

1. قم بعمل `Clone` أو تحميل للمستودع (Repository).
2. قم ببناء المشروع محلياً باستخدام الأمر:
   ```bash
   npm run build
   ```
3. قم برفع المجلد الناتج `dist` إلى مستودع جيت هاب وتفعيل **GitHub Pages** عليه.
4. للحصول على مفتاح Gemini API مجاني لتشغيل البوت في موقعك العام، تفضل بزيارة [Google AI Studio](https://aistudio.google.com/) والحصول على الـ Key وضعه في خانة الإعدادات بالموقع لتجربة خارقة وسريعة!
